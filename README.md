# Bot de Automação com PyAutoGUI

## Sobre o projeto

Este projeto foi utilizado para estudos de automação em Python utilizando a biblioteca PyAutoGUI.

O código não foi desenvolvido inteiramente por mim. A base foi obtida a partir de um tutorial, e utilizei o projeto como forma de aprender conceitos de programação, automação de teclado e mouse, reconhecimento de imagens na tela e estruturação de scripts.

Durante o estudo, analisei o funcionamento do código, corrigi erros de sintaxe, realizei testes e busquei compreender a lógica utilizada.

## Objetivos de aprendizado

- Entender estruturas condicionais (`if`)
- Praticar laços de repetição (`while`)
- Aprender a criar e utilizar funções
- Trabalhar com parâmetros e retorno de funções
- Utilizar reconhecimento de imagens na tela com PyAutoGUI
- Automatizar ações de teclado e mouse
- Desenvolver habilidades de leitura e interpretação de código existente

## Tecnologias utilizadas

- Python
- PyAutoGUI
- Keyboard

## Observações

Este projeto possui finalidade exclusivamente educacional e foi utilizado como prática de programação e automação.

Meu foco principal durante o desenvolvimento foi compreender a lógica do código e aprimorar meus conhecimentos em programação.

## Aprendizados

A principal lição deste projeto foi perceber a importância da lógica de programação. Embora eu tenha iniciado estudando a automação em si, grande parte das dificuldades encontradas estava relacionada à compreensão da lógica por trás do código.

O projeto serviu como motivação para aprofundar meus estudos em algoritmos, lógica de programação e desenvolvimento em Python.
